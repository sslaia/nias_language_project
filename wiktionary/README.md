# Proyek Kamus Nias on Wiktionary

## Standar penamaan audio files untuk pengucapan:

nia-[dialek-]kata.ogg

nia = kode bahasa

## Penamaan dialek: 
- **teng** = Nias Tengah
- **sel** = Nias Selatan
- **bar** = Nias Barat
- **tim** = Nias Timur
- **ut** = Nias Utara
